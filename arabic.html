Let's start learning Arabic by mastering the alphabet!

أ ب ت ث ج ح خ د ذ ر ز س ش ص ض ط ظ ع غ ف ق ك ل م ن ه و ي

أ [alif]

is for أب (father)

ب [baa’]

is for بيت (house)

ت [taa’]

is for تفاحة (apple)

ث [thaa’]

is for ثعبان (snake)

ج [jeem]

is for جمل (camel)

ح [haa’]

is for حصان (horse)

خ [khaa’]

is for خبز (bread)

د [dal]

is for دم (blood)

ذ [thal]

is for ذهب (gold)

ر [raa’]

is for رمل (sand)

ز [zain]

is for زهرة (flower)

س [seen]

is for سماء (sky)

ش [sheen]

is for شمس (sun)

ص [Ssaad]

is for صحراء (desert)

ض [ddad]

is for ضفدع (frog)

ط [tt- uh]

is for طائر (bird)

ظ[tthhad]

is for ظل (shadow)

ع [‘eain]

is for عين (eye)

غ [ghain]

is for غيمة (cloud)

ف [faa’]

is for فم (mouth)

ق [khaaf]

is for قمر (moon)

ك [kaaf]

is for كتاب (book)

ل [lam]

is for لعبة (toy)

م [meem]

is for ماء (water)

ن [noon]

is for نجم (star)

ه [haa’]

is for هاتف (telephone)

و [waw]

is for وردة (rose)

ي [yaa’]

is for يد (hand)
